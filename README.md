# Personal Finance Tracker

A desktop-based personal finance management application built with Python. This project allows users to track their income and expenses, organize transactions into categories, and monitor their financial activity through a simple and user-friendly interface.

## Features

- Add and manage income and expense transactions
- Categorize spending
- Track financial history
- Store data using SQLite database
- Generate spending summaries
- Visualize financial data with charts
- Export transaction data
- Clean and intuitive GUI interface

## Technologies Used

- Python
- SQLite
- CustomTkinter / Tkinter
- Matplotlib
- CSV data handling

## Purpose

The goal of this project was to create a practical finance management tool while improving skills in Python programming, database management, GUI development, and data visualization.

## Future Improvements
